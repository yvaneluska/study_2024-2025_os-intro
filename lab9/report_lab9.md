# 2. Теоретические сведения

## 2.1 Понятие командной оболочки

Командная оболочка — это интерфейс взаимодействия пользователя с операционной системой, позволяющий выполнять команды и управлять файлами.

## 2.2 Midnight Commander

Midnight Commander (mc) — файловый менеджер для Linux/Unix с псевдографическим интерфейсом.

Основные возможности:
- двухпанельный интерфейс
- управление файлами
- встроенный редактор
- поддержка горячих клавиш

---

# 3. Интерфейс программы

## 3.1 Структура
- левая и правая панели
- строка меню (F9)
- командная строка
- функциональные клавиши

## 3.2 Функциональные клавиши

F1 — помощь  
F3 — просмотр  
F4 — редактирование  
F5 — копирование  
F6 — перемещение  
F7 — создание каталога  
F8 — удаление  
F10 — выход  

---

# 4. Режимы работы панелей

- стандартный режим
- режим информации
- режим дерева каталогов

Дополнительно:
- Ctrl+U — смена панелей
- Ctrl+O — скрытие панелей
- Ctrl+X D — сравнение каталогов

# 5. Меню программы

## 5.1 Меню «Файл»

Позволяет:

Позволяет:

## 5.3 Меню «Настройки»

Содержит:

- настройки панелей
- подтверждения операций
